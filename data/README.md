# Datos semanales del dashboard E140
   
   Esta carpeta contiene los archivos JSON con los datos de cada semana.
   
   Cada archivo sigue el formato `data-sXX.json` donde XX es el número de semana.
   Ej: `data-s36.json`, `data-s37.json`...
   
   Los archivos se generan automáticamente con el botón "📂 Cargar Semana" del dashboard.
   No modifiques a mano estos archivos.
